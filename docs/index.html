VirtualDoc App

DSI2021

VirtualDoc App es un proyecto que traza una ayuda en el manejo y control de pacientes para muchos doctores y ambientes de servicio médico. Los clientes de este proyecto procuran de este proyecto una ayuda en el registro de pacientes, consultas, acceso a la información médica que deben disponer de sus pacientes como medicamentos, enfermedades tratadas anteriormente o actuales y además tener un fácil acceso a estos a través de la web y de app móvil.
